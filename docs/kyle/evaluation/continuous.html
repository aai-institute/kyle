compute_ECE(dirichlet_fc: kyle.sampling.fake_clf.DirichletFC, conditioned='full', **kwargs)[source]

Computes theoretical ECE of dirichlet_fc Fake Classifier conditioned on full confidence vector, conditioned on the confidence in prediction or conditioned on each class confidence separately (see 1 for further details)

Parameters

  • dirichlet_fc – Dirichlet fake classifier to calculate ECE for

  • conditioned – Quantity to condition ECE on

  • kwargs – passed to integrator function

Returns

  • If conditioned on full confidence vector returns: result, abserr, (further scipy.nquad output)

  • If conditioned on the confidence in prediction returns: result, abserr, (further scipy.quad output)

  • If conditioned on each class separately returns: List of num_classes+1 entries. First entry contains average of all “i-class ECEs”. Subsequent entries contain results for each “i-class ECE” separately: result, abserr, (further scipy.quad output)
